Pink Figurehead - Logarithmic Function

Transformations

I took this photo in a makeup shop at Sunway Pyramid. I chose this photo because it is my favourite makeup shop in Malaysia. I really enjoy makeup and buying new products there, so I always visit this shop whenever I go to Sunway Pyramid.

To model the pink figurehead appearing in the photograph, I first identified that the shape closely resembles a logarithmic function. Therefore, I selected the parent function \( y=\log(x) \) as the starting point.

From observing the orientation of the curve in the image, I noticed that the graph decreases from left to right rather than increasing. This indicated a reflection across the y-axis, so I modified the function to \( y=\log(-x) \) to match the direction of the graph.

Initially, I found it difficult to determine the correct transformation because the visible part of the curve was very small and looked different from the parent logarithmic graph. Since the graph appeared very flat and was positioned away from the original parent function, it was difficult to directly compare the two shapes.

After further analysis, I realised that the function had undergone several transformations. The negative sign inside the logarithm represents a reflection across the y-axis, causing the graph to decrease from left to right.

Next, I analysed the leading coefficient \(0.1\). Since there is no negative sign before the coefficient, there is no reflection across the x-axis. Because \(0.1\) is less than \(1\), the graph undergoes a vertical compression. This transformation makes the graph significantly flatter than the parent function.

I experimented with many different values in order to closely match the curve shown in the photograph. Larger values than \(0.1\) made the graph too steep, while smaller values flattened the graph excessively. Therefore, \(0.1\) produced the best fit for the photograph.

After determining the overall shape, I applied translations to move the graph into the correct position. I shifted the graph horizontally to the right so that the vertical asymptote aligned with its position in the image. Then, I applied a vertical translation upward so the curve passed through important points in the photograph.

The graph initially appeared too low, so I translated it upward by \(1.2\) units. This resulted in the final function:

\[ y=0.1\log\bigl(-(x-0.6)+1.2\bigr) \]

This equation closely models the shape shown in the photograph.

Domain and Range

The domain of my function is:

\[ D=\{x\in\mathbb{R}\mid 0.43\le x\le 0.58\} \]

I could not fit the equation to the shape in the photograph while including the entire range of x-values, so I restricted the domain to the interval that best matched the visible curve.

The parent function \(y=\log(x)\) has the domain:

\[ x>0 \]

and the range:

\[ y\in\mathbb{R} \]

This means the function is defined only for positive x-values and can produce any real number. The parent logarithmic function also has a vertical asymptote at \(x=0\), where the function is undefined.

After applying transformations, my function also has a vertical asymptote. However, the restricted interval used in my model is relatively far from the asymptote, so the discontinuity does not affect the part of the graph being analysed.

Since the function is continuous on the restricted interval, the range can be determined by evaluating the function at the endpoints of the domain. By substituting \(x=0.43\) and \(x=0.58\), I obtained approximate y-values of \(1.12\) and \(1.03\).

Therefore, the approximate range is:

\[ R=\{y\in\mathbb{R}\mid 1.03\le y\le 1.12\} \]
